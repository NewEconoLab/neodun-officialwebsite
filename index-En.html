NEODUN is a hardware wallet for storing private keys of cryptocurrencies like Bitcoin, Ethereum, and NEO. It ensures the security of private keys and cryptocurrency payments cryptographically. The content of each transaction is displayed on the device screen for review and confirmation by the user.

Your money only belongs to you after you control your private key. NEODUN eliminates the risk of theft of private keys through physical isolation and software encryption. The details of each transaction on NEODUN will be shown to you in 100%, the transaction can be valid only after you personally confirm it.

NEODUN is used to manage the cryptocurrency private key, which can be used to transfer, collect payments and sign smart contracts. To ensure the security of the private key generation, NEODUN cancelled the automatic generation of the private key. The user can generate the private key through the wallet or use the NEODUN driver to import the private key offline.
The NEODUN device does not support private key backup. You need to back up the private key before importing the private key into NEODUN. The wallet that supports NEODUN will support an encrypted backup function, and we will also provide offline backup tools.
When restoring, you need to use the backup file and the backup password. After inserting NEODUN, you can import it according to the wallet or drive prompt.

FAQ

01
Why do I need NEODUN？
When you hold a digital currency, you can't avoid the threat from hackers, whether we put the digital currency on the exchange or on a personal computer. The safest way now is to use a cold wallet to store the private key of digital currency in a disconnected environment. However, the cold wallet is only safe for storage, and it needs to be reconnected to the Internet again when the transaction is conducted. The process is complicated and risky. NEODUN is a cold wallet that solves these transaction process problems, allowing you secure storage and transaction.
02
How to back up NEODUN?
For security reasons, private keys in NEODUN will only be destroyed and will not be exported. Before you import the private key into NEODUN, you can use our offline tool to generate an encrypted backup file, and then save this backup file to an offline device such as a USB flash drive. You can also use your own method, such as a paper wallet, a brain wallet, etc., to record the private key, and then import it offline when necessary.
03
Why does NEODUN import private keys from the outside instead of generating them internally?
NEODUN is positioned as a hardware wallet for private key management and needs to be compatible with multiple private keys. Therefore, from the standpoint of the randomness of the private key generation and the convenience of operation, importing methods after external generation is better than the wallet does itself.. The PC has a lot of random information (such as mouse and keyboard movement, PC number, etc.) to ensure the security of the generated private key, and we also ensure the security of the import process through encryption.
04
NEODUN is open source, does it mean that it is less secure?
We believe that open source and transparency can achieve stronger security. Through open source, our products can benefit from the supervision, review, and suggestions of more professionals, so that our algorithm can stand the validation of more people, and ultimately win more users’ trust.
05
What happens if NEODUN is lost?
NEODUN has a strong encryption protection mechanism so that someone who accidentally picks up a NEODUN lost by another cannot get the private key inside unless he also knows your login password. But still, in this case, please transfer the money in your wallet to somewhere else in case someone steals your NEODUN and then steals your login password.
06
What to do when forgetting the password of NEODUN？
For the security of your private key, we do not provide the function of recovering your password. If you forget the password. The wallet will be reset and delete all the data inside once you enter the password incorrectly five times. Then you just need to re-import the private key and set the password.
07
Since NEODUN has to import private keys from the outside, how can I ensure that the software that generates my private key is trustworthy?
Please use wallets which have links on our official website to generate, backup and import private keys. These wallets are also open source and support NEODUN. Of course, if you have wallets from trusted sources, you can also use them to generate private keys and manually import private keys through our driver. Our driver can be run offline.
08
Can NEODUN be used on Android or IOS phones?
We are developing drivers for other platforms, and we can only use NEODUN on the windows platform for the time being.
09
Can secondhand NEODUN be used?
In extreme cases, fraudsters can spend huge amounts of manpower to create a fake hardware and software package in the NEODUN shell to steal users’ private keys, which ordinary consumers cannot identify. So when you want to use NEODUN from an unknown source, please update the firmware which will automatically detect if NEODUN is changed internally.
10
What do we need to pay attention when you keep NEODUN?
Protect your NEODUN as you protect your mobile phone.
11
Can NEODUN be used in an insecure network environment?
NEODUN can operate in any network environment and use environment when it is normally used. However, before the private key enters NEODUN, it cannot be protected by NEODUN. Therefore, we recommend that you do not create or import private keys in insecure environments, such as public computers and computers that are not protected by anti-virus software.
